To add a unsubscribe from WooCommerce in the EDIT ACCOUNT DETAILS page.

== Description ==

To account details edit page of woocommerce the user to add a button to leave on their own.
It does not appear in the administrator.

[Caution]
When a user deletes an account all such as coupons and points will be removed.
